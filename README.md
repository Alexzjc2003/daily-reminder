# daily-reminder

## Introduction

Daily-reminder, or dr, is a command-line reminder designed to use with .bashrc.

For more intros, check [Intro](./INTRO.md)

## Build dr

This should require go installed. Under root of this repo

```sh
./scripts/build.sh
```

and `dr` will be in `build`

## Try dr

Please refer to [Intro](./INTRO.md) for usage.

In [`scripts/tests`](./scripts/tests) lies some test scripts. Some may require [`tm`](https://github.com/Alexzjc2003/schweizer-messer/file/c/tm.c). You may ignore them if you don't care, and may write your own tests.
